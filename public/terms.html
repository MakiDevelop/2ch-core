一、服務性質

2ch.tw（以下簡稱「本平台」）是一個由使用者自主發文的匿名討論平台（User-Generated Content, UGC）。

本平台不對使用者發表的內容進行事前審查，所有內容均由發文者自行負責。

二、使用者責任與義務

2.1 內容責任

使用者對自己發表的所有內容負完全法律責任。

  • 您發表的內容不得違反中華民國法律（包括但不限於刑法、民法、著作權法等）
  • 您發表的內容不得侵害他人權利（包括但不限於名譽權、隱私權、著作權）
  • 您同意不發表任何違反本平台版規的內容

2.2 禁止行為

以下行為嚴格禁止，違者將被立即處理：

  1. 違法內容：教唆犯罪、散布違法資訊、兒少不當內容、恐怖主義相關內容
  2. 個資外洩（Doxxing）：公開或暗示他人個人資料（姓名、地址、電話、身分證字號等）
  3. 明確誹謗：針對可被具體辨識的個人指控犯罪或不法行為，且無公共利益必要
  4. 惡意攻擊：洗版、灌水、垃圾訊息、惡意騷擾
  5. 商業操作：廣告、行銷、刻意帶風向、組織性灌票

2.3 資料收集同意

使用者發表內容即表示同意本平台記錄以下資訊：

  • IP 位址（用於平台管理與法律調查）
  • 裝置資訊（User-Agent）
  • 發文時間與內容

上述資料僅用於平台管理與配合法律調查，將於 90 天後自動刪除（涉及法律調查者除外）。

三、平台權利與責任範圍

3.1 平台提供的服務

  • 提供匿名討論空間
  • 提供合理的內容管理（刪除明顯違法或違規內容）
  • 配合合法的司法調查

3.2 平台的權利

本平台保留以下權利，無需事前通知或說明理由：

  • 刪除、隱藏或修改任何內容
  • 鎖定或關閉特定討論串
  • 封鎖特定 IP 或使用者
  • 暫停或終止服務
  • 修改本使用者條款

3.3 平台的責任限制

本平台不對使用者發表的內容承擔責任。

  • 本平台不保證能即時發現或處理所有違規內容
  • 本平台不介入觀點對錯的爭議，僅處理行為越界
  • 本平台不保證服務不中斷、不出錯
  • 使用者因發表內容產生的法律問題，由使用者自行承擔
重要說明：本平台採「合理管理」原則，這意味著：
  • 我們會處理明顯違法的內容，但無法保證即時處理
  • 我們不對使用者的言論正確性或合法性負責
  • 管理行為以「平台安全與討論品質」為優先

四、資料保護與隱私

4.1 資料收集

本平台會記錄以下資訊：

  • IP 位址：以 SHA-256 雜湊後的形式顯示於前台（無法反查）
  • 真實 IP：僅保存於資料庫，用於管理與法律調查
  • User-Agent：裝置與瀏覽器資訊

4.2 資料保存期限

  • 一般發文：真實 IP 與 User-Agent 保存 90 天後自動刪除
  • 涉及法律調查：依法律要求延長保存期限
  • IP 雜湊值：永久保存（用於前台顯示，無法反查真實 IP）

4.3 資料使用

本平台僅在以下情況使用您的資料：

  • 平台管理（處理違規行為）
  • 配合司法機關合法調查
  • 系統安全維護

本平台不會主動公開或販售使用者資料。

五、法律配合與司法調查

當收到合法的司法機關要求時，本平台有權且有義務：

  • 提供特定發文者的 IP 位址與裝置資訊
  • 提供發文內容與時間記錄
  • 配合調查所需的其他必要資訊

本平台僅提供「最小必要範圍」的資料，不提供無關使用者的資訊。

匿名不代表無法追查。若您發表違法內容，司法機關可透過 IP 位址追查到您。您將為自己的違法行為承擔完全的法律責任，包括但不限於刑事責任與民事賠償。
